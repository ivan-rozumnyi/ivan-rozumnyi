# Привіт!

Цей репозиторій — моє місце для навчання HTML та CSS. 
Мета — створити дружню, безпечну та інклюзивну спільноту для всіх, хто заходить на цей репозиторій.

---

## Наша обіцянка
Ми, учасники та автори, прагнемо зробити участь у спільноті безпечною для всіх, незалежно від віку, статі, досвіду, національності, релігії тощо.
Ми обіцяємо діяти так, щоб підтримувати відкриту, дружню та інклюзивну спільноту.

## Наші стандарти
### Позитивна поведінка включає:
- Доброзичливість та емпатію
- Повагу до інших думок
- Конструктивний фідбек
- Визнання своїх помилок та навчання на них
- Дбати про благо спільноти

### Неприпустима поведінка включає:
- Тролінг, образи, переслідування
- Публікацію чужої особистої інформації без дозволу
- Сексуальні натяки або домагання
- Будь-яка неприйнятна поведінка в професійному середовищі
